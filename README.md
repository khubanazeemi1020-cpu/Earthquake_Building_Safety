# Mechanics Behind Earthquakes and Building Safety

## Project Overview
This repository contains all files and resources related to the study of earthquake mechanics and building safety. The project explores how seismic forces affect structures, the principles of structural stability, and methods to design buildings that can withstand earthquakes. The goal is to analyze and understand the physics behind building safety and provide practical insights for civil engineering applications.

## Repository Contents
The repository is organized as follows:

- **Reports/**  
  Contains the main project report detailing concepts, research, and findings related to earthquake mechanics and building safety.

- **Diagrams/**  
  Includes illustrations, structural diagrams, and visualizations that explain forces acting on buildings during seismic events.

- **Simulations/**  
  Contains any simulation files, calculations, or models used to analyze structural stability, stress distribution, or seismic impact.

- **README.md**  
  This file providing project overview and guidance for users.

## Key Features
- Analysis of seismic forces on buildings  
- Principles of structural design for earthquake resilience  
- Simulation of stress and strain on building structures  
- Visual representation of forces and safety measures
